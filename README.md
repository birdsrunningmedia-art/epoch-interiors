# Epoch Interiors — Portfolio Mockup

A modern, responsive portfolio website for a mock interior design company called **Epoch Interiors**.
Built to showcase high-end interior design projects using **Next.js**, **Tailwind CSS**, and **shadcn/ui**.

This is a frontend-focused mockup project meant for design presentation, UI practice, and portfolio display.

---

## ✨ Preview

This website features:
- Clean, minimalist design inspired by modern interior studios
- Elegant typography and spacing
- Fully responsive layout

---

## ⚙️ Tech Stack

- **Next.js 14+ (App Router)**
- **Tailwind CSS**
- **shadcn/ui**
- **Lucide Icons**
- **Framer Motion** (for smooth animations)
